# BuildMyPC

An AI voice-based website that will help you build your PC

# Features:
- Voice-based interaction
- PC parts lister
- Benchmark my PC
- What to upgrade?
- Suggested phrases per chat

# Tested Environments:
- Windows
- Fedora

## Disclaimer
The first 3 releases were made before the repo was created. That is the reason why the releases have very short time gaps.
